### 파이썬 순열과 조합 

```
* Python permutation

- 소스코드 

from itertools import permutations

a = [1,2,3]
permute = permutations(a,2)
print(list(permute))

결과
[(1,2),(1,3),(2,1),(2,3),(3,1),(3,2)]
```
