## 서울의 기온 데이터 분석하기 
```
- 기상자료개방포털 => 기후통계분석 => 기온분석

- 책에 적혀 있는 1904 01 01 ~ 2019 01 17
```
### CSV 파일 이란?
```  
- 정부에서 운영하는 공공 데이터 포털이 제공하는 일반적인 파일 형식
```

### CSV 파일에서 데이터 읽어오기 
```
- csv.reader()
  csv 파일에서 데이터를 읽어오는 함수
  
- csv.writer()
  csv 파일에 데이터를 저장하는 함수
```
